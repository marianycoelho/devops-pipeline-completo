# Projeto DevOps Completo

Projeto pessoal para estudo prático de DevOps.

## Tecnologias
- GitHub Actions (CI/CD)
- Docker (Containerização)
- Terraform (Infrastructure as Code)
- Python

## O que foi feito
- Aplicação simples em Python
- Containerização da aplicação com Docker
- Pipeline CI/CD acionado a cada push
- Execução automática do container
- Definição de infraestrutura via Terraform
- Simulação de arquitetura cloud

## Fluxo
1. Push no repositório
2. Pipeline CI/CD é executado
3. Imagem Docker é construída
4. Container é executado automaticamente
5. Infraestrutura é definida via Terraform
6. Aplicação pronta para rodar em ambiente cloud

## Objetivo
Demonstrar conhecimentos básicos de DevOps e automação de infraestrutura.
